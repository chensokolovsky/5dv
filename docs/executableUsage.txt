5dv - executable usage:

use one of these options:

1) 5dv enc [path to an uncompressed 5dvu file] [5dv path to save the compressed file]
2) 5dv dec [path to a 5dv file] [path to save the uncompressed file]
3) 5dv create [type] [destination path for the 5dvu file]. available types :spheres
4) 5dv display [element 1] [optionl:element 2]. These must be uncompressed 5dvu files
5) 5dv full [folder name]. Runs the full process: create, encode, decode, shhow. All files will get created in that folder. Make sure the folder exists and empty

for more details see: http://5dv.org/quickStart.text
